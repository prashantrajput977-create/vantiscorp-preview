# VantisCorp — Site Mockup

Static HTML/CSS preview of vantiscorp.com revisions. Built for review with the CEO before WordPress handoff.

## Deploy to Render

1. Render dashboard → New → Static Site
2. Connect this GitHub repo
3. **Build command:** _(leave empty)_
4. **Publish directory:** `.`
5. Deploy

The site is a static bundle — no build step needed.

## Pages

- `index.html` — Homepage (mirrors live vantiscorp.com layout)
- `platform.html` — Platform overview (four agents)
- `booking-agent.html`, `compliance-agent.html`, `finance-agent.html`, `servicing-agent.html` — Agent detail pages
- `integrations.html` — Integration partners and stack
- `about.html` — Company, mission, leadership
